# Devoir Appliqué – Projet HTML/CSS

## Présentation

Ce projet regroupe plusieurs exercices pratiques pour apprendre et appliquer les bases du HTML et du CSS. Il s'agit d'une mini-application web composée de plusieurs pages, chacune illustrant des notions différentes (structure, tableaux, formulaires, positionnement, etc.).

## Structure du projet

- **index.html** : page d'accueil du projet
- **pages/** : contient les pages d'exercices
    - exercice-2.html : Tableaux et gestion de données
    - exercice-3.html : Formulaire de contact
    - exercice-4.html : Positionnement (Flexbox & Grid)
- **assets/** : ressources partagées
    - css/ : feuilles de style pour chaque exercice et style global
    - img/ : images utilisées dans les exercices
    - fonts/ : polices éventuelles
    - consigne/ : consignes ou documents de référence
    - w3c/ : fichiers de validation ou d'accessibilité

## Technologies utilisées
- HTML5 (balises sémantiques, formulaires, tableaux...)
- CSS3 (sélecteurs, flexbox, grid, responsive, etc.)

## Objectifs pédagogiques
- Structurer une page web avec des balises sémantiques
- Créer et styliser des tableaux
- Réaliser un formulaire accessible et ergonomique
- Mettre en page des éléments avec Flexbox et Grid
- Organiser un projet web de façon professionnelle
